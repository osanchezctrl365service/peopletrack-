Centro de Ayuda
👋Esta sección explica a detalle cada módulo de PeopleTrack. Está pensada para que cualquier usuario pueda entender la plataforma sin necesidad de capacitación previa.
📊 Dashboard

El dashboard muestra un resumen del equipo asignado. Verá tarjetas con el avance promedio, objetivos en riesgo y reuniones pendientes. Haga clic en el nombre de cada colaborador para ver su detalle.

🎯 Objetivos

Los objetivos tienen dos tipos: Core (definidos por la empresa para todos) y Personales (acordados con cada colaborador). Cada objetivo tiene una ponderación que suma 100% por persona. Puede registrar check-ins de avance periódicos.

📈 Plan de Carrera

Define hitos de desarrollo para cada colaborador desde su rol actual hasta el rol objetivo. Cada hito tiene un porcentaje de avance y fecha de revisión. El progreso total se calcula automáticamente.

⭐ Competencias

Se evalúan competencias genéricas (para todos), de liderazgo (para líderes) y personales. La escala es de 0 a 10 y se mapea a 4 niveles: No desarrollado, Sigue trabajando, Viene Bien, Viene Súper.

🤝 Reuniones 1:1

Registre reuniones semanales, mensuales o trimestrales con cada colaborador. Las notas se acumulan cronológicamente (bitácora). Puede enviar la minuta directamente por email haciendo clic en el botón 📧 junto a cada persona.

⚙ Configuración

Configure la conexión SQL, Azure AD para SSO, la API de Claude AI para asistencia inteligente, y el umbral de alertas. En Importación puede descargar el template CSV para cargar usuarios y sus relaciones de liderazgo.
